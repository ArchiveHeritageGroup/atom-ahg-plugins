### Quick Start

1. **Data Migration**: Admin > Data Migration > Upload CSV/Excel
2. **Donor Agreements**: Admin > Donor Management > New Agreement
3. **Spectrum Workflows**: Record > Spectrum > Start Workflow
4. **Data Ingest**: Admin > Ingest > New Session
