## Plugins

### Themes
- **arAHGThemeB5Plugin** - Bootstrap 5 theme with modern UI

### Extensions (Coming Soon)

## Requirements

- AtoM 2.10.x
- atom-framework 1.0.0+
- PHP 8.3+
